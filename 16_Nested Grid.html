Nested Grid

Un ejemplo sencillo de anidar una cuadrícula dentro de otra. Cualquier área de cuadrícula puede convertirse en una cuadrícula en sí misma, estableciendo display: grid y luego definiendo las filas y columnas.

        .contenedor {
            display: grid;
            grid-template-columns: repeat(4, [col] 150px);
            grid-template-rows: repeat(2, [row] auto);
            ...
        }  
        
        .d {
            grid-column: col 3 / span 2;
            grid-row: row 2;

            display: grid;
            grid-template-columns: 1fr 1fr;
            ...
        }

        .e {
            grid-column: 1 / 3;
            grid-row: 1;
        }

        .f {
            grid-column: 1;
            grid-row: 2;
        }

        .g {
            grid-column: 2;
            grid-row: 2;
        }
