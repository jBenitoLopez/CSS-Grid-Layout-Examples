Para definir una subcuadrícula solo en las filas, utiliza el valor "subgrid" en un elemento de cuadrícula del contenedor como valor de "grid-template-rows".

En este ejemplo, luego he creado pistas de cuadrícula para las columnas en la subcuadrícula usando "grid-template-columns: 1fr 1fr 1fr". Estas pistas de columna se ajustarán dentro del área donde se coloca la cuadrícula anidada, al igual que con una cuadrícula anidada regular.


    .contenedor {
        display: grid;
        grid-template-columns: 150px 100px 150px 100px;
        grid-template-rows: repeat(3, minmax(100px, auto));
        ...
    }
    .a {
        grid-column: 1 / 3;
    }
    .b {
        grid-column: 4;
    }
    .c {
        grid-column: 1; grid-row: 2 / 4;
    }
    .d {
        grid-column: 2 / 5; grid-row: 2 / 4;

        display: grid;
        grid-template-columns: 1fr 1fr 1fr;
        grid-template-rows: subgrid;
        ...
    }
    .e {
        grid-row: 1 / 3; grid-column: 1;
    }
    .f {
        grid-row: 1; grid-column: 2 / 4;
    }
    .g {
        grid-row: 2; grid-column: 3;
    }
