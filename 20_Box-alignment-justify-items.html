Box alignment justify-items

He utilizado una imagen de fondo de cuadrícula para mostrar la cuadrícula en este ejemplo. He definido 4 áreas de cuadrícula de igual tamaño a la izquierda que cubren un área cuadrada dentro de tres líneas de fila y columna.

Estoy utilizando la propiedad justify-items con un valor de center. Esto centra el contenido de todos los elementos de la cuadrícula.

Cada una de las cajas a la izquierda tiene un área de cuadrícula de 3 columnas y 3 filas (estamos contando el espacio entre col/row).

La propiedad align-items se utiliza para alinear el contenido dentro de cada área de cuadrícula.

Otros valores de align-items son:

  • stretch
  • start
  • end
  • center
