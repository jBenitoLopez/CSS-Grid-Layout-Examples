Para definir una subcuadrícula solo en las columnas, utiliza el valor "subgrid" en un elemento de cuadrícula del contenedor como valor de "grid-template-columns".

Las filas ahora pueden ser definidas con una lista de pistas normal de "grid-template-rows" o se puede usar una cuadrícula implícita como es habitual.

Cualquier cuadrícula creada para las filas será solo para la subcuadrícula, y la fila de la cuadrícula principal que contiene la subcuadrícula crecerá para ser lo suficientemente grande como para contener el contenido (asumiendo que tiene un tamaño automático u otro tamaño basado en el contenido).
