Puedes superponer elementos en la cuadrícula utilizando la propiedad z-index para controlar el orden en el que se apilan. En este ejemplo, estan las cajas dispuestas en una cuadrícula y la caja F es la última en el orden de origen y está posicionada en la cuadrícula de manera que se superpone parcialmente con la caja D.

Sin ninguna propiedad z-index, se mostrará encima de la caja D. Sin embargo, puedo agregar un z-index y controlar la posición, lo cual funciona como esperarías si has utilizado la propiedad z-index con elementos posicionados.


        grid-template-columns: repeat(5, [col] 100px);
        grid-template-rows: repeat(3, [row] auto);

        .caja {
            background-color: #444;
            color: #fff;           
            z-index:10;
        }

        .f {
            grid-column: col 3 / span 3;
            grid-row: row 2;
            z-index: 20;

            background-color: rgba(243, 54, 130, 0.5);
        }
