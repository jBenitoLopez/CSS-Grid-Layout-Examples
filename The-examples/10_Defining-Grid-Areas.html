Defining Grid Areas

Podemos crear áreas con nombre en la cuadrícula para colocar contenido. Para hacer esto, primero asignamos elementos en nuestro marcado a un Área de Cuadrícula utilizando la propiedad grid-area.

Luego, podemos usar la propiedad grid-template-areas para describir dónde deben ubicarse estos elementos en la cuadrícula.

Repetir el nombre de un área indica que el área abarca esa pista de la cuadrícula. Usar un punto (.) o una secuencia como .... indica una pista vacía. Por ejemplo, el footer deja el primer espacio vacío.

    
        grid-template-columns: 32% 32% 32%;
        grid-template-areas:
            "header header header"
            "sidebar content content"
            "....... footer footer"
        ;
