The auto-fill keyword with named grid lines

Este ejemplo contiene tantas pistas de columna de 100 píxeles como quepan en el contenedor (en el ejemplo, esto es el viewport) y se están nombrando "col". Luego podemos posicionar los elementos de la cuadrícula utilizando líneas y extensiones con nombres.


    .contenedor {
        grid-template-columns: repeat(auto-fill, [col] 100px);
        grid-template-rows: repeat(auto-fill, [row] auto);
    }
    .a {
        grid-column: col / span 2;
        grid-row: row;
    }
    .b {
        grid-column: col 3 / span 2;
        grid-row: row;
    }
    .c {
        grid-column: col;
        grid-row: row 2;
    }
    .d {
        grid-column: col 2 / span 3;
        grid-row: row 2;
    }
    .e {
        grid-column: col / span 4;
        grid-row: row 3;
    }
