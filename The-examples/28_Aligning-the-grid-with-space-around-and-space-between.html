Aligning the grid with space-around and space-between

En este ejemplo, se crea una cuadrícula de cuatro pistas de columna. Las pistas tienen tamaños absolutos y en total son más pequeñas que el área del contenedor de la cuadrícula. Esta vez he utilizado: 

            
                align-content: space-around
                justify-content: space-between

Esto significa que el espacio adicional se distribuye alrededor de las pistas y nuestro espacio deseado de 10 píxeles entre columnas obtiene más espacio. Sin embargo, las áreas que abarcan más de una pista también obtienen este espacio adicional cuando cruzan un espacio entre columnas.


    .contenedor {
        ...
        grid-template-columns: repeat(4, 80px);
        grid-template-rows: repeat(3, 100px);
        align-content: space-around;
        justify-content: space-between;
    }
    .caja {
        ...
    }
    .item1 {
        grid-column: 1 / 5;
    }
    .item2 {
        grid-column: 1 / 3;
        grid-row: 2 / 4;
    }
    .item3 {
        grid-column: 3 / 5;
    }

Si los bordes exteriores de la cuadrícula no coinciden con los bordes del contenido del contenedor de la cuadrícula (por ejemplo, si no hay columnas de tamaño flexible), las pistas de la cuadrícula se alinearán dentro del recuadro de contenido según las propiedades justify-content y align-content del contenedor de la cuadrícula.
