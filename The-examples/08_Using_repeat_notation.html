Line-based placement shorthand (grid-area)

En este ejemplo repetimos las mismas definiciones para crear nuestra cuadrícula con líneas con nombres.

Podríamos ahorrar algo de escritura utilizando la notación "repeat". Los valores para "repeat" son el número de veces que deseas que se repita la expresión y luego la expresión en sí.

ver 07_Line_Based-Placement_Named-Lines-with-spans

        grid-template-columns: repeat(4, [col] 100px ) ;
        grid-template-rows: repeat(3, [row] auto  );
