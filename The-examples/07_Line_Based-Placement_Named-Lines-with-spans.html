Line-based placement named lines width spans

Puedes darles el mismo nombre a las líneas y luego usar la palabra clave "span" para dirigirte a líneas con un nombre específico.

Esto es realmente útil si deseas crear una cuadrícula compleja con múltiples pistas de contenido y espacios entre ellas.

grid-template-columns: [col] 100px [col] 100px [col] 100px [col] 100px ;
.b { grid-column: col 3 / span 2 ; }
