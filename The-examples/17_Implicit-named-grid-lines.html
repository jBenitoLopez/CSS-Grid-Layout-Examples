Implicit named grid lines

Cuando se utilizan áreas con nombre, se crean líneas implícitas con nombre agregando "-start" y "-end" al área con nombre que has definido. Se pueden utilizar de la misma manera que las líneas que has nombrado explícitamente.

        .contenedor {
            display: grid;
            grid-template-columns: 120px 120px 120px;
            grid-template-areas:
                "header  header  header"
                "sidebar content content";
            ...
        }

        .content {
            grid-area: content;
        }

        .header {
            grid-area: header;
        }

        .sidebar {
            grid-area: sidebar;
        }

        .overlay {
            z-index: 10;
            grid-column: content-start / content-end;
            grid-row: header-start / content-end;
        }

La propiedad grid-template-areas genera nombres de líneas asignadas implícitamente a partir de las áreas de cuadrícula con nombre en la plantilla. Para cada área de cuadrícula con nombre "content", se crean cuatro nombres de líneas asignadas implícitamente: dos llamados "content-start", que nombran las líneas de inicio row-start y column-start de la área, y dos llamados "content-end", que nombran las líneas row-end y column-end para el fin de fila y de fin de columna del área.

Estos nombres de líneas asignadas implícitamente se comportan como cualquier otro nombre de línea, excepto que no aparecen en el valor de grid-template-rows/grid-template-columns. Incluso si se define un nombre de línea explícitamente asignado con el mismo nombre, los nombres de líneas asignadas implícitamente son simplemente más líneas con el mismo nombre.
