Box alignment align-items

Se utilizo una imagen de fondo de cuadrícula para mostrar la cuadrícula en este ejemplo. He definido 4 áreas de cuadrícula de igual tamaño a la izquierda que cubren un área cuadrada dentro de tres líneas de fila y columna.

Se esta usando la propiedad align-items con un valor de center. Esto centra el contenido de todos los elementos de la cuadrícula.

En la caja a y e se le añadio el estilo align-self: stretch;

La propiedad align-items se utiliza para alinear el contenido dentro de cada área de cuadrícula.

Otros valores de align-items son:

  • stretch
  • start
  • end
  • center
