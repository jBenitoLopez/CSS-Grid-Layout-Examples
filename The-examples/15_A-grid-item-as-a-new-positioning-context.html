A grid item as a new positioning context

Puedes posicionar elementos de forma absoluta dentro de un área de la cuadrícula. En este ejemplo, se utilizado la propiedad position: relative en la área .content. Ahora se puede posicionar las cuatro imágenes de flechas utilizando posicionamiento absoluto dentro de esa área, y funciona como esperarías.

Header
Sidebar
Content
The four arrows are inline images inside the content area.
Footer

        .contenedor {
            display: grid;
            grid-template-columns: 200px 200px 200px;
            grid-template-areas:
                "header  header  header"
                "sidebar content content"
                "footer  footer  footer";
        }
    
        .content {
            grid-area: content;
            position: relative;
            background-color: tomato;
            color: #fff;
        }
    
        .topleft {
            position: absolute;
            top: 0;
            left: 0;
        }
        
        .topright {
            position: absolute;
            top: 0;
            right: 0;
        }
        
        .bottomleft {
            position: absolute;
            bottom: 0;
            left: 0;
        }
        
        .bottomright {
            position: absolute;
            bottom: 0;
            right: 0;
        }
