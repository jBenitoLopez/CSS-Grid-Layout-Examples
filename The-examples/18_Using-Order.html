Using Order

Eje. 1

La cuadrícula admite la propiedad order, que también se encuentra en Flexcaja. Si estás posicionando explícitamente elementos de la cuadrícula, entonces el orden afectará el orden de pintado y, por lo tanto, el orden en el que los elementos se apilan cuando no se ha aplicado ningún z-index.

Si estás utilizando la colocación automática (auto-placement), entonces la propiedad order afectará cómo se colocan los elementos en la cuadrícula. En este ejemplo, todas las cajas tienen un order de 1, por lo que siguen siendo posicionadas en el orden del DOM. Sin embargo, la caja1 tiene un order de 3 y la caja2 tiene un order de 2. Estas cajas tienen un valor de order más alto, por lo que se posicionan después de todas las cajas con un valor de order de 1.
