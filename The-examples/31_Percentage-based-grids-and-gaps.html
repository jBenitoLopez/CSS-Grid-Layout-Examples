Percentage based grids and gaps

Una cuadrícula con un ancho del 90%. Seis pistas de columna de un 10% cada una, y cinco pistas de espacio entre columnas de un 2% cada una.

La propiedad grid-gap controla las columnas y filas, pero como la cuadrícula no tiene altura, el espacio entre filas se resuelve a 0.

Si le damos a la cuadrícula una altura, hay algo para que el 2% sea un porcentaje de. Así que obtenemos un espacio entre filas.

Una cuadrícula con un ancho del 90%. Seis pistas de columna de un 10% cada una y cinco pistas de espacio entre columnas de un 2% cada una.

La propiedad grid-gap controla las columnas y filas, pero como la cuadrícula no tiene altura, el espacio entre filas se resuelve a 0. Por lo tanto, en este caso, solo se verá el espacio entre columnas, pero no habrá espacio entre filas debido a la falta de altura definida para la cuadrícula.


            .contenedor {
                display: grid;
                grid-template-columns: repeat(6, 10%);
                width: 90%;
                ...
