minmax() in auto-fill repeating tracks

En este ejemplo se crea una cuadrícula que contiene tantas pistas de columna de 200 píxeles como quepan en el contenedor, y el espacio restante se comparte por igual entre las columnas. En la función minmax(), el primer valor es el tamaño mínimo que deseo que tengan mis pistas, y el segundo valor es el máximo.

Al usar 1fr como valor máximo, el espacio se distribuye de manera equitativa.


        .contenedor {
            grid-template-columns: repeat(auto-fill, minmax(200px, 1fr));
        }

La función minmax() define un rango de tamaño mayor o igual a "min" y menor o igual a "max". Si el valor de "max" es menor que el valor de "min", entonces el valor de "max" se establecerá como el valor de "min" (esencialmente, obtendríamos minmax(min, min)). Cuando se usa como valor máximo, un valor establece el factor de flexión de la pista; sin embargo, es inválido como valor mínimo.

minmax(min, max)
